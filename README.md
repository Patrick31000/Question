Q1: Joseph Carl Robnett Licklider peut être considéré comme le fondateur d'Internet car il a été le premier a
décrire les interactions sociales possibles avec un réseau d'ordinateurs en 1962 pour la DARPA (Defense Advanced Research Projects Agency). Mais le réseau Internet n'est entré officiellement en service en 1969. L'objectif initial n'était pas commercial, il était de pouvoir maintenir les télécommunications en cas d'attaque (éventuellement nucléaire).
